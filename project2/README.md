# Number Guessing Game

A simple Python game where the computer generates a random number between 1 and 100, and the user tries to guess it.

## Features
- Random number generation
- User input handling
- Hint messages (Higher/Lower)
- Counts the number of attempts

## Technologies Used
- Python
- Random Module

## How to Run
python main.py
Guess the number: 50
Higher number please

Guess the number: 75
Lower number please

Guess the number: 63
You have guessed the number correctly in 3 attempts
